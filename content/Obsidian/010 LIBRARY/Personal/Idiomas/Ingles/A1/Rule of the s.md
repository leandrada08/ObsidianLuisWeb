## Rule of the s

| Letters      | Rule                                  |
| ------------ | ------------------------------------- |
|              | Agrego 's': book-->books              |
| H,S,O,X      | Agrego 'es': kiss-->kisses            |
| Consonante+y | Cambio y por 'ies': family-->families |
| F o FE       | Cambio f por 'es': half-->hales       |
| Irregulares  | No siempre se sigue la regla          |
